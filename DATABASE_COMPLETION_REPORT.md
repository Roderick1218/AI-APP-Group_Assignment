## 🚀 使用建议

### 最佳实践
1. **具体询问**: 问具体的情况而不是泛泛的问题
2. **场景描述**: 描述您的具体旅行场景
3. **多轮对话**: 可以追问更详细的信息
4. **分类查询**: 按照不同类别（航班、酒店、餐饮等）查询
